# Jigen Insight Design System

- Use Element Plus as the interaction primitive layer.
- Favor rounded surfaces, dark cards on light atmospheric backgrounds, and soft contrast.
- Keep component typography aligned with global tokens in src/assets/styles/global/variables.less.
- Prefer CSS grid/flex layouts and avoid layout-only wrapper components.
- All user-facing text must come from i18n keys.
- Ensure interactive custom elements show cursor: pointer.
- Preserve responsive behavior for desktop and mobile breakpoints.
